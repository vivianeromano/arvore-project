# Arvore-Project

This project was created to reproduce the gitgub commits board
## Environments
Put the following keys (with their correct values) inside your `.env` file:

```
REACT_APP_BASE_URL=http://localhost:3001
```
## Available Scripts

In the project directory, you can run:
### `npm -g json-server`
Install as a global dependency. This lib will work with a server.
 
In a separate terminal run the command: 
```
json-server --watch db.json --port 3001
```
### `yarn`
This command will install all the dependencies that the project needs to function
### `yarn start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### `yarn test`

Launches the test runner in the interactive watch mode.\
See the section about [running tests](https://facebook.github.io/create-react-app/docs/running-tests) for more information.
